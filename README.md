# Swift-Number-Abbrevation-Comma
Just a quick extension of Int for abbreviating with commas.

Format a number and adjust based on the number of length: X / XX / XXX / X,XXXX / XX.Xk / XXX.xk / X.XM.

```
Input: -999 -> -999
Input: -1284 -> -1284
Input: 598 -> 598
Input: 1000 -> 1,000
Input: 9940 -> 9,940
Input: 9980 -> 9,980
Input: 10500 -> 10.5K
Input: 39900 -> 39.9K
Input: 99880 -> 99.9K
Input: 399880 -> 0.4M
Input: 999898 -> 1M
Input: 999999 -> 1M
Input: 1456384 -> 1.5M
Input: 12383474 -> 12.4M
```
